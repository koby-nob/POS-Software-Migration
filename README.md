# POS-Software-Migration

<h2>Description </h2>

    - This project entails the process of migrating Legacy Point of Sale (POS) system from windows to Android system.
    
        
<h2>Operating System Used </h2>

     - Windows 10 Pro (Production Enviroment PC)
         
<h2>Remote Tools </h2>
  
    - Air-viewer 
    - Team-Viewer
    - Bomgar 
    - LogMeIn
